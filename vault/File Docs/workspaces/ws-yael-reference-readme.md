# yael/reference/README.md

**שייך ל:** יעל

## מה הקובץ עושה
קובץ הסבר לתיקיית הדוגמאות (reference) של יעל. מסביר שלתיקייה זו מוסיפים קבצי `.md` עם דוגמאות לטקסטים בסגנון הבית, ושיעל קוראת את כל הקבצים בתיקייה בתחילת כל משימת שכתוב כדי לדייק את הסגנון. מגדיר מוסכמות: שם קובץ תיאורי (למשל `example-blog-post.md`), כותרת בראש הקובץ המסבירה את הפורמט/ערוץ, והערה קצרה למה הדוגמה מייצגת את הסגנון.
