## מה הקובץ עושה
זהו קובץ ההוראות של ראובן, מנכ"ל צוות הסוכנים. הוא מגדיר את תפקיד ראובן בקבלת בקשות מהמשתמש וניתובן לסוכן המתאים (יעל לכתיבה, יובל לתמונות, חן למחקר). הקובץ כולל את מבנה התיקיות של הפרויקט, טבלאות ניתוב לפי מילות מפתח בעברית ובאנגלית, ותהליכי עבודה מלאים (חיבור יעל↔יובל למאמר עם תמונות, וזרימת חן→יעל→יובל לתוכן חדש מהרשת). בנוסף מוגדרים כללי עבודה קבועים: שימוש ב-vault כזיכרון ארוך-טווח וקומיט גיט אוטומטי בסיום משימה.
