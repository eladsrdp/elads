# yael/reference/example-short-use-cases.md

**שייך ל:** יעל

## מה הקובץ עושה
דוגמת בית בתיקיית reference של יעל — ארבעה פוסטים קצרים שמדגימים אינטגרציות ווצאפ-פריוריטי (בוט כמו אפליקציה, GPS ואימות מיקום, אוטומציה של פעולות ידניות, שליחת מסמכים חשבונאיים). הדוגמה מייצגת את סגנון הבית: מבנה קצר-קצר, פתיחה עם הסיטואציה, הסבר הבעיה והפתרון, גוון טכני בלי להיות יבש, וסיום בשאלה/הנעה קלה. משמשת את יעל ללימוד וחיקוי הסגנון לפני שכתוב.
