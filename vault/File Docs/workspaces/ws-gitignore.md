## מה הקובץ עושה
מגדיר אילו קבצים מוחרגים ממעקב גיט. מחריג את `.claude/settings.local.json` (הגדרות מקומיות אישיות למכונה), את `.env` (סודות סביבה — מפתחות API אמיתיים), ואת תיקיית `Output/` (תוצרים מגוּלמים של יעל שאינם נשמרים בבקרת גרסאות). חשוב לאבטחה: ההחרגה של `.env` מבטיחה שמפתחות אמיתיים לא נדחפים ל-git, בעוד `.env.example` כן נשמר במעקב כתבנית בלי ערכים.
