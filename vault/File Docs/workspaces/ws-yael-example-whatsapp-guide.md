# yael/reference/example-whatsapp-guide.md

**שייך ל:** יעל

## מה הקובץ עושה
דוגמת בית בתיקיית reference של יעל — פוסט הסבר טכני-עסקי מקיף על ווצאפ ביזנס, מתאים לבלוג/לינקדאין. מסביר בשכבות את ההבדל בין ווצאפ רגיל, ווצאפ ביזנס וחיבור API (רשמי ולא רשמי), כולל מגבלות, עלויות, חלון 24 שעות ותבניות הודעה. הדוגמה מייצגת את סגנון הבית: פתיחה עם שאלות אמיתיות של לקוחות, הסבר משכבות, כנות לגבי מגבלות, שפה חברותית, משפטים קצרים והומור עצמי. משמשת את יעל ללימוד הסגנון.
