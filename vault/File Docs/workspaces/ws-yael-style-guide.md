# yael/style-guide.md

**שייך ל:** יעל

## מה הקובץ עושה
מדריך הסגנון המרכזי של יעל, שנגזר מניתוח 5 טקסטים של המחבר המקורי. יעל קוראת אותו לפני כל משימת שכתוב. המדריך מגדיר את טון הכתיבה (מקצועי-חברותי, ישיר, אנושי, כן), מבנה הטקסט (פתיחה עם שאלות, גוף בשכבות, סיום קצר), מבנה פסקאות (קצר מאוד, שורה אחת למחשבה), ניסוחים מועדפים ואסורים (טבלת "במקום X נכתוב Y", מילים אהובות, מבנים אסורים פורמליים), כלים טכניים (כותרות, הדגשה, מקפים), וטיפול במספרים ומחירים. כולל גם תבנית HTML מלאה (RTL, נקייה) לפורמט ה-Output.
