# .env.example

**שייך ל:** כלל-הפרויקט

## מה הקובץ עושה
תבנית להגדרת משתני הסביבה של הצוות, נשמרת במעקב גיט (להבדיל מ-`.env` עצמו). מכיל את שמות המפתחות בלבד, בלי ערכים: `ANTHROPIC_API_KEY` (מפעיל את ראובן ואת כל הסוכנים הטקסטואליים מבוססי Claude), `OPENAI_API_KEY` (יצירת תמונות עבור יובל דרך skill gpt-image-gen), ו-`TAVILY_API_KEY` אופציונלי (חיפוש/מחקר ברשת עבור חן). המשתמש מעתיק את הקובץ ל-`.env` וממלא את הערכים האמיתיים.

> **הערה אבטחה:** קובץ `.env` אכן קיים בשורש הפרויקט, מוחרג מגיט דרך `.gitignore`, ומכיל את הערכים האמיתיים של המפתחות הללו. הוא אינו מתועד כאן ואין לקרוא ממנו ערכים גלויים.
