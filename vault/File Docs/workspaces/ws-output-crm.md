# Output/מאמר CRM.md + .html

**שייך ל:** יעל

## מה הקובץ עושה
הגרסה הסופית של מאמר ה-CRM, בתיקיית `Output/` — שני פורמטים: MD ו-HTML. יעל לקחה את טקסט המקור הגולמי מ-`Content/מאמר CRM.txt` ושכתבה אותו לפי מדריך הסגנון: כותרת "מה זה CRM — ולמה עסק שצומח לא יכול בלעדיו", פסקאות קצרות מאוד, מבנה משכבות מהפשוט למורכב, ומקטע שאלות-ותשובות בסוף. המאמר לא כולל תמונות (אין placeholders), ולכן יובל לא הופעל עבורו. תיקיית Output מוחרגת מגיט.
