## מה הקובץ עושה
סקיל המנחה את סגירת עבודת הפיתוח כשהמימוש הושלם וכל הבדיקות עוברות. מציג אפשרויות מובנות לשילוב העבודה — מיזוג (merge), פתיחת PR או ניקוי — ועוזר להחליט בין מסלולי הסיום.

קבצים פנימיים:
- `SKILL.md` — האפשרויות המובנות לסיום ענף וכללי ההחלטה ביניהן.
