**שייך ל:** תצורת Claude Code / סקילים

## מה הקובץ עושה
Wrapper לקריאת OpenAI Images API ליצירת תמונה מ-prompt טקסטואלי. הסקיל מקבל prompt ונתיב פלט, קורא ל-API (מודל `gpt-image-2` בלבד — קבוע, אין להחליפו), מקבל תמונה ב-base64 ושומר אותה כקובץ PNG. מופעל על ידי יובל בכל בקשת תמונה. דורש `OPENAI_API_KEY` הנטען מ-`.env` בשורש — המפתח אינו מופיע בקוד או בתיעוד. מתואר מסלול ראשי (curl + jq) ו-fallback ב-Python כש-jq/base64 חסרים, וכן שלב אימות שהקובץ קיים וגודלו גדול מאפס.

קבצים פנימיים:
- `SKILL.md` — ההנחיות המלאות: שם המודל הקבוע, פרמטרי הבקשה, מסלול curl ו-fallback ב-Python, ואימות פלט.
