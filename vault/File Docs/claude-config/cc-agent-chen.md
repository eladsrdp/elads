## מה הקובץ עושה
מגדיר את **חן**, חוקרת הרשת של הצוות. הסוכן מופעל לבקשות חיפוש, מחקר ומציאת מקורות (טריגרים בעברית ובאנגלית). חן מחפשת באינטרנט מקורות אמיתיים ועדכניים עם לינקים לאימות, מסננת לפי קריטריוני איכות (מקורות ראשוניים, פרסומים מקצועיים, תאריך עדכני), ושומרת את המקור הנבחר כקובץ ב-`Content/` עם לינק למקור בראשו. היא מנהלת לוג זיכרון ב-`chen/Memory/searches.md` כדי לא לחפש פעמיים את אותו דבר ב-30 הימים האחרונים. ארכיטקטונית היא לא קוראת ישירות ליעל — רק מכינה את הקרקע ומדווחת לראובן. הכלים שלה: WebSearch, WebFetch, Read, Write, Edit, Glob, Grep.
