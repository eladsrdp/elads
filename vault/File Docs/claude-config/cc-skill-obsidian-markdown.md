## מה הקובץ עושה
סקיל ליצירה ועריכה של Obsidian Flavored Markdown — ההרחבות הייחודיות של אובסידיאן מעל CommonMark/GFM: wikilinks, embeds, callouts, properties (frontmatter), הערות, תגיות ועוד. מופעל בעבודה על קבצי .md באובסידיאן או כשהמשתמש מזכיר wikilinks, callouts, frontmatter, תגיות או embeds. כולל זרימת יצירת note ותחביר מלא לקישורים פנימיים.

קבצים פנימיים:
- `SKILL.md` — תחביר ה-Obsidian Markdown וזרימת יצירת ה-note.
