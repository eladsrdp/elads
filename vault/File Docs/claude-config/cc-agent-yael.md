## מה הקובץ עושה
מגדיר את **יעל**, כותבת התוכן של הצוות. הסוכן מופעל למשימות שכתוב, עריכה, ניסוח מחדש, תרגום וסיכום של מאמרים ופוסטים (טריגרים בעברית ובאנגלית). יעל קוראת מאמר גלם מתיקיית `Content/`, לומדת את `yael/style-guide.md` ואת דוגמאות הבית ב-`yael/reference/`, ומשכתבת בסגנון הבית בלי לשנות עובדות, נתונים או ציטוטים. בזמן הכתיבה היא מסמנת מקומות לתמונה באמצעות placeholder בפורמט `{{IMAGE_NEEDED: "..."}}`, ושומרת שתי גרסאות (MD ו-HTML) ב-`Output/`. הכלים שלה: Read, Write, Edit, Glob, Grep — בלי גישה לרשת, ליצירת תמונות או להפעלת סוכנים אחרים.
