## מה הקובץ עושה
סקיל ליצירה וניהול של קבצי `.base` באובסידיאן — תצוגות דמויות-מסד-נתונים של notes עם פילטרים, נוסחאות (formulas) ותצוגות table/cards/list/map. מופעל כשרוצים לבנות base, לתשאל notes לפי תגית/property/תאריך, או להגדיר פריסת תצוגה. כולל זרימת עבודה, סכמת YAML וכללי ציטוט (quoting) למניעת שגיאות.

קבצים פנימיים:
- `SKILL.md` — זרימת העבודה, סכמת ה-`.base`, וכללי ה-YAML והתצוגות.
