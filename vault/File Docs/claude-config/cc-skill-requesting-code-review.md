## מה הקובץ עושה
סקיל לבקשת סקירת קוד: שיגור subagent סוקר שמקבל הקשר מדויק ומתוכנן בלבד (לא היסטוריית הסשן), כדי לתפוס בעיות לפני שהן מתפשטות. מופעל בסיום משימות, מימוש פיצ'רים גדולים, או לפני מיזוג, כדי לוודא שהעבודה עומדת בדרישות.

קבצים פנימיים:
- `SKILL.md` — מתי וכיצד לבקש סקירה ואיזה הקשר להעביר לסוקר.
- `code-reviewer.md` — הגדרת/prompt הסוכן הסוקר.
