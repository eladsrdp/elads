**שייך ל:** תצורת Claude Code / סקילים

## מה הקובץ עושה
סקיל ליצירה, עריכה ואימות של סקילים לפני הפצתם. מרכז שיטות עבודה מומלצות, עקרונות שכנוע לכתיבת הנחיות אפקטיביות, וטכניקה לבדיקת סקילים באמצעות subagents. כולל כלי המחשה לגרפים והמרתם לתמונות.

קבצים פנימיים:
- `SKILL.md` — תהליך כתיבת הסקיל ואימותו.
- `anthropic-best-practices.md` — שיטות עבודה מומלצות לכתיבת סקילים.
- `persuasion-principles.md` — עקרונות לניסוח הנחיות משכנעות.
- `testing-skills-with-subagents.md` — בדיקת סקילים בעזרת subagents.
- `graphviz-conventions.dot` + `render-graphs.js` — מוסכמות גרפים והסקריפט להמרתם לתמונות.
- `examples/CLAUDE_MD_TESTING.md` — דוגמה לבדיקת CLAUDE.md.
