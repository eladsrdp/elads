# .claude/skills/karpathy-guidelines/

**שייך ל:** תצורת Claude Code / סקילים

## מה הקובץ עושה
סקיל ובו הנחיות התנהגות לצמצום טעויות נפוצות של מודלי שפה בכתיבת קוד, מבוסס על תצפיותיו של Andrej Karpathy. ארבעה עקרונות: לחשוב לפני קוד (לציין הנחות, לא להסתיר בלבול), פשטות תחילה (מינימום קוד, בלי ספקולציות), שינויים כירורגיים (לגעת רק במה שצריך), וביצוע מונחה-מטרה (להגדיר קריטריוני הצלחה ולאמת). מטה לזהירות על פני מהירות.

קבצים פנימיים:
- `SKILL.md` — ארבעת העקרונות וההנחיות ליישומם.
