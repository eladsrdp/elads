## מה הקובץ עושה
מגדיר את **יובל**, מעצב התמונות של הצוות. הסוכן מופעל לבקשות ליצירת תמונה, ציור או איור (טריגרים בעברית ובאנגלית). המטרה העליונה שלו היא עקביות ויזואלית בין כל התמונות בפרויקט: הוא סורק את תמונות ההשראה ב-`yuval/reference/`, מחלץ סגנון, פלטת צבעים וקומפוזיציה, ומנסח prompt שמשלב בין הבקשה לסגנון הבית. את התמונה הוא יוצר דרך הסקיל `gpt-image-gen` (מודל `gpt-image-2`), שומר PNG וקובץ `.txt` תאום עם ה-prompt ב-`yuval/outputs/`, ומאמת שהקובץ קיים וגודלו גדול מאפס. הכלים שלו: Read, Write, Bash, Glob.
