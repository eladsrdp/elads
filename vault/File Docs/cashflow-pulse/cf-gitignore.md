# cashflow-pulse/.gitignore

## מה הקובץ עושה
רשימת ההתעלמות של git — ה-gitignore הסטנדרטי של תבנית Vite: קבצי לוג, ‏node_modules, תיקיות build‏ (dist, dist-ssr), קבצי `*.local`, ותיקיות/קבצי עורכים (vscode, idea, DS_Store וכו'). הערה: אין בו עדיין כלל ל-`.env` — רלוונטי אם יתווספו בעתיד סודות או מפתחות סביבת פיתוח.
