## מה הקובץ עושה
דף ה-HTML היחיד של ה-SPA. מגדיר את המסמך כעברית RTL‏ (`lang="he" dir="rtl"`), viewport מותאם מובייל (ללא זום, viewport-fit=cover), צבע theme כהה, קישור ל-favicon ול-manifest של ה-PWA, וכותרת "תזרים — Cash Flow Pulse". מכיל את `<div id="root">` וטוען את `src/main.tsx` כמודול.
