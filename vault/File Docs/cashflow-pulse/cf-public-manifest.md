# cashflow-pulse/public/manifest.webmanifest

## מה הקובץ עושה
‏Web App Manifest שמאפשר להתקין את האפליקציה כ-PWA במסך הבית: שם "תזרים — Cash Flow Pulse", שפה עברית וכיווניות RTL, תצוגת standalone (ללא כרום דפדפן), ורקע/theme בצבע הכהה של האפליקציה (#0f172a). מערך ה-icons ריק כרגע — נקודה להשלמה אם רוצים אייקון התקנה תקין.
