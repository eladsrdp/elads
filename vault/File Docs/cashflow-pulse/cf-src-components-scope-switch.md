# cashflow-pulse/src/components/ScopeSwitch.tsx

**שייך ל:** cashflow-pulse / שכבת ה-UI (קומפוננטות)

## מה הקובץ עושה
מתג עליון בצורת pill להחלפה בין הכיס העסקי לביתי. קומפוננטה נשלטת (value/onChange) שמקבלת `ScopeId` ומציגה שני כפתורים — "עסקי" ו"ביתי" — כשהנבחר מודגש ברקע בהיר. מוצג רק במסכי הדופק וספר התנועות (לא בהעברות והגדרות, שאינם תלויי-כיס).
