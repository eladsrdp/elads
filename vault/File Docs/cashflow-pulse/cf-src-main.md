## מה הקובץ עושה
נקודת הכניסה של האפליקציה: מייבא את עיצוב הבסיס (`index.css`), יוצר React root על האלמנט `#root` מתוך `index.html`, ומרנדר את `App` בתוך `StrictMode`. זה הקובץ שה-script של Vite טוען ראשון.
