## מה הקובץ עושה
ניווט תחתון קבוע (sticky) בסגנון אפליקציית מובייל, עם ארבע לשוניות: דופק, ספר, העברות והגדרות — כל אחת עם אייקון אמוג'י ותווית בעברית. מגדיר ומייצא את הטיפוס `Tab` שמשמש גם את `App` לניהול המסך הפעיל. הלשונית הפעילה מודגשת בירוק (emerald).
