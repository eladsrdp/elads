# cashflow-pulse/src/components/forms.tsx

**שייך ל:** cashflow-pulse / שכבת ה-UI (קומפוננטות בסיס, טפסים)

## מה הקובץ עושה
ערכת רכיבי הטפסים המשותפת: `Field` (תווית מעל שדה), `TextInput` ו-`Select` (עטיפות דקות עם class עיצוב אחיד — כהה, מעוגל, פוקוס ירוק), ו-`PrimaryButton` (כפתור פעולה ראשי ירוק עם מצב disabled). כל הטפסים באפליקציה — הוספת צפוי, העברות, קבועים, הגדרות כיס וייבוא — בנויים מהרכיבים האלה.
