## מה הקובץ עושה
מסך "דיווחים" — נקודת האישור והשליחה לפריוריטי: מציג את הטיוטות הממתינות (כולל כאלה שנכשלו עם הודעת השגיאה שלהן), כפתור "אשר ושלח לפריוריטי" עם ספירה וסה"כ שעות, והיסטוריית הדיווחים שכבר סונכרנו מקובצת לפי תאריך. אחרי סנכרון מציג סיכום (כמה נשלחו / נכשלו), ובכשל רשת מודיע שהטיוטות נשמרו לניסיון חוזר.
