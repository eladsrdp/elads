## מה הקובץ עושה
קובץ הנעילה של npm לכל ה-monorepo — מקבע גרסאות מדויקות של כל התלויות בשלושת ה-workspaces ‏(shared, server, client) כדי שהתקנות יהיו שחזוריות בכל סביבה. נוצר ומתוחזק אוטומטית על ידי `npm install`; לא עורכים אותו ידנית.
