## מה הקובץ עושה
כרטיס הטיימר — הרכיב המרכזי במסך "היום". במצב מנוחה: כפתור גדול "התחל טיימר" שפותח את בורר המשימות. במצב ריצה: שם המשימה והפרויקט, שעון רץ בפורמט שעות:דקות:שניות עם נקודה ירוקה מהבהבת, וכפתור "עצור ושמור טיוטה". אם הטיימר רץ מעל 16 שעות (isStale) מציג אזהרה "נראה ששכחת אותו" עם אפשרות לבטל בלי לשמור.
