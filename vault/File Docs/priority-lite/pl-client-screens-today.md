## מה הקובץ עושה
מסך "היום" — מסך הבית של האפליקציה: כרטיס הטיימר למעלה, כרטיס "סה״כ היום" שמחבר את הדיווחים השמורים עם הטיימר הרץ בזמן אמת, רשימת הדיווחים של היום (ממוינים מהחדש לישן, עם עריכה ומחיקה לטיוטות), וכפתור צף (+) לפתיחת דיווח ידני. אותו מודאל משמש גם ליצירה וגם לעריכת דיווח קיים.
