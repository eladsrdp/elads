## מה הקובץ עושה
שורת דיווח בודדת — מציגה שם משימה, פרויקט, תאריך (אופציונלי), שעות התחלה/סיום אם קיימות, הערה, משך מפורמט, ו-chip סטטוס צבעוני (טיוטה / נשלח… / בפריוריטי ✓ / שגיאה). לטיוטות ושגיאות מציגה כפתורי עריכה ומחיקה, ולדיווח שנכשל בסנכרון — את הודעת השגיאה. בשימוש בשני מסכים: "היום" ו"דיווחים".
