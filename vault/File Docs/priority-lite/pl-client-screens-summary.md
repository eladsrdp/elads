## מה הקובץ עושה
מסך "סיכום" — בורר טווח (היום / השבוע / החודש), כרטיס סה"כ גדול עם ציון כמה מהשעות עדיין בטיוטה, ופירוט מקובץ לפי פרויקט עם המשימות שלו וסימון כוכבית למשימות שכוללות שעות בטיוטה. כל החישוב מקומי דרך useRangeSummary — עובד גם offline.
