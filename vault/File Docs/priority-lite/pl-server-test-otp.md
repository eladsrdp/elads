## מה הקובץ עושה
בדיקות vitest ללוגיקת ה-OTP, עם SQLite בזיכרון ועובד בדיקה פיקטיבי שנטען לפני כל בדיקה. מכסה: נרמול טלפון (פורמט מקומי, קידומת ‎+972, רווחים ומקפים, דחיית מספרים לא תקינים), happy path של בקשת קוד, דחיית מספר שאינו ב-whitelist, ‏rate limit (חסימה אחרי המכסה ושחרור אחרי החלון), תפוגת קוד ומגבלת ניסיונות — תוך שימוש בפרמטר ה-now המוזרק כדי לשלוט בזמן.
