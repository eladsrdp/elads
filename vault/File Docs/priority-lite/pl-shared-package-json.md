## מה הקובץ עושה
מגדיר את החבילה `@priority-lite/shared` — workspace קטן שכל תפקידו לייצא את קובץ הטיפוסים המשותפים. ה-exports מפנה ישירות ל-`./src/types.ts` (בלי build — השרת רץ עם tsx והקליינט עם Vite, ושניהם צורכים TypeScript גולמי).
