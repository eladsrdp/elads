# priority-lite/server/scripts/smoke.ts

**שייך ל:** priority-lite / שרת / סקריפטים

## מה הקובץ עושה
בדיקת עשן ידנית מול פריוריטי אמיתי דרך ה-OData adapter. פקודת `read` (בטוחה) מריצה חיפוש משימות, שליפת משימה בודדת וקריאת דיווחי שעות של עובד בטווח תאריכים. פקודת `write <DOCNO>` כותבת דיווח בדיקה של 15 דקות עם הערה "נא להתעלם" וקוראת אותו חזרה לאימות — מיועדת לחברת טסט בלבד. פרטי החיבור נקראים מ-.env, ומזהה העובד מ-SMOKE_EMP.
