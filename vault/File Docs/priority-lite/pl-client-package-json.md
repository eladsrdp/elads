## מה הקובץ עושה
מגדיר את החבילה `@priority-lite/client` ואת התלויות שלה: React 19 ‏(+react-dom), ‏Dexie ‏(+dexie-react-hooks) ל-IndexedDB מקומי, והטיפוסים המשותפים. כלי הפיתוח: Vite 8 עם plugin-react, ‏Tailwind CSS v4 (דרך @tailwindcss/vite), ‏TypeScript ו-vitest. סקריפטים: dev, ‏build ‏(tsc -b + vite build), ‏preview ו-test.
