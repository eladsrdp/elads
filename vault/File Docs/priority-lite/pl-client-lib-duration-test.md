## מה הקובץ עושה
בדיקות vitest לעוזרי המשך: פורמט fmtMin (כולל 0), כל ענפי הפענוח של parseDuration — פורמט שעות:דקות, מספרים כשעות עד 24, מספרים שלמים כדקות מעל 24, ודחיית קלט לא תקין (ריק, דקות מעל 59, טקסט, אפס, ערכים מעל יממה) — וחישוב הפרשי שעות ב-diffMinutes.
