# priority-lite/client/index.html

**שייך ל:** priority-lite / קליינט / שלד האפליקציה

## מה הקובץ עושה
דף ה-HTML היחיד של ה-SPA — מוגדר `lang="he" dir="rtl"` כך שכל האפליקציה עברית מימין לשמאל. טוען את פונט Heebo מ-Google Fonts, מקשר את ה-manifest וה-icon של ה-PWA, קובע theme-color כהה ו-viewport מותאם נייד (כולל viewport-fit=cover), וטוען את `/src/main.tsx` כמודול. ‏Vite משתמש בו כנקודת הכניסה ל-build.
