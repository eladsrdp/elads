# priority-lite/client/src/index.css

**שייך ל:** priority-lite / קליינט / עיצוב

## מה הקובץ עושה
ה-CSS הגלובלי — מייבא את Tailwind v4 (שורת import אחת, בלי קובץ config) ומוסיף בסיס: ערכת צבעים כהה (color-scheme: dark, רקע ‎#0f172a), פונט Heebo, גובה מלא ל-root, וביטול tap-highlight בנייד. מחלקת העזר `ltr-nums` מציגה מספרים משמאל לימין גם בתוך טקסט RTL — בשימוש בכל תצוגות השעות והמספרים באפליקציה.
