## מה הקובץ עושה
אייקון האפליקציה — SVG וקטורי של שעון (עיגול עם מחוגים) בירוק-אמרלד על רקע כהה מעוגל, בהתאם לערכת הצבעים של האפליקציה. משמש גם כ-favicon בדף ה-HTML וגם כאייקון ה-PWA בהתקנה למסך הבית (מוצהר ב-manifest עם sizes="any").
