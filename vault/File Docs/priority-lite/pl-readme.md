## מה הקובץ עושה
מסמך הפתיחה של הפרויקט: מסביר ש-Priority Lite היא אפליקציית PWA בעברית (RTL) לדיווחי שעות ומשימות מעל Priority ERP, עם טיוטות מקומיות שנשלחות רק אחרי אישור. מתאר את מבנה ה-monorepo ‏(shared / server / client), הוראות הרצה בפיתוח (npm run seed + npm run dev, בלי צורך ב-.env), הרצת בדיקות, ושלבי החיבור לפריוריטי אמיתי (M6) דרך mapping.ts ומשתני סביבה.
