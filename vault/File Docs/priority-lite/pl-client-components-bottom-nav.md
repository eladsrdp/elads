# priority-lite/client/src/components/BottomNav.tsx

## מה הקובץ עושה
ניווט תחתון קבוע בסגנון אפליקציית מובייל — ארבעה טאבים (היום / דיווחים / סיכום / הגדרות) עם אייקון ותווית, הדגשת הטאב הפעיל בירוק, ו-badge כתום על טאב "דיווחים" עם מספר הפריטים הממתינים. מגדיר ומייצא את הטיפוס `Tab` שמשמש את App לניהול הניווט.
